# デモサイト公開状況

## 公開設定

デモサイトはCodex Sitesへ限定公開している。

公開スラッグは`open-loop-inbox`である。

サイトの表示名は`Open Loop Inbox`である。

アクセスは所有者だけに限定している。

プラットフォームが発行するホスト名にはアカウントの名前空間が含まれる場合がある。

個人情報を含まないURLを固定するには、専用の独自ドメインを接続する必要がある。

## デモの範囲

デモサイトはサンプルデータを使い、Open Loopの回収、照合、判断、擬似Receiptの体験を提供する。

外部サービスへの書込みは行わない。

実際のCodex履歴を扱うCompanionとPluginは、各利用者のローカル環境で動かす。

## サムネイル

共有用のOG画像は設定していない。

デプロイ用サムネイルも設定していない。

外部サービスが以前に取得した共有プレビューは、そのサービスのキャッシュ期限まで表示される場合がある。

## 確認

公開前に本番ビルドを成功させた。

画面操作は手動で確認する。
